## To run the project locally

1. Clone the repository

```bash
git clone https://github.com/lakshaybhushan/wth-backend
```

2. Install the dependencies

```bash
bun install
```

3. Run the server

```bash
bun run dev
```

4. To deploy the project to Cloudflare Workers, run

```bash
bun run deploy
```
